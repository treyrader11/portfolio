My Portfolio: Frontend Web Development

Below are most of my projects for the Thinkful FEWD career path course. All source code is available here.

Click on the images and image titles to read more.


Streetfighter App

This project brings together DOM manipulation, events and effects, while building a Street Fighter themed interactive web page. The purpose is to practice using the jQuery API and get a better sense of what the workflow might look like when using jQuery for interactive effects on a web page.

The Shopping List

My first project I designed and coded from scratch. The purpose is for students to practice designing and wireframing, appling HTML, styling with CSS, and using jQuery events to add interactive behavior.

Hot or Cold

Applying functions and jQuery to build a "Hot or Cold" game. In this game, the computer randomly selects a number between 1 and 100, and the player then tries to guess the number. The player gets feedback for each guess - "hot" if their guess was close, and "cold" if their guess was far.

Cajun Music Quiz

An interactive quiz app built from scratch that focuses on object literals.

Tic Tac Toe

A Tic tac Toe game designed and coded from scratch using jQuery, and later refactored so to be more object-oriented.

LYRUS Music App

An app that brings together everything I learned about HTML, CSS, JavaScript, jQuery, and AJAX in a single capstone project, an API hack. It is meant for the student to gain experience researching RESTful APIs and designing client-side applications to interface with them, and see a project through from ideation to implementation.
